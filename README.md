# GovTech 2026 Intern Listings

A lightweight, fast, client-side static web application to browse GovTech's 2026 internship roles without suffering through a messy Excel spreadsheet.

## Live Site
[https://pebblepaw.github.io/govtech2026-intern-listings/](https://pebblepaw.github.io/govtech2026-intern-listings/)

## Features
- **Filter & Search:** Search by keywords, and filter by Role (Sheet), Division, and Level.
- **Favourites:** Star jobs (★) to save them locally using your browser's `localStorage`.
- **Compare:** Select up to 3 roles and compare their prerequisites, learning outcomes, and descriptions side-by-side.
- **Zero Tracking:** 100% static HTML/JS with no external tracking or analytics.
